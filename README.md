# 不要受伤 (Don't Get Hurt)
Minecraft Fabric 模组，版本 1.21.11
**注意，所有代码均为AI生成，此mod仅为娱乐，请不要在您的极限生存存档等重要存档安装此mod！！！**
## 功能

当玩家受到**任何伤害**时（包括跌落、溺水、岩浆、着火等），会立即在玩家周围 5 格范围内随机生成以下 6 种生物组合之一（等概率）：

1. 凋零 ×1
2. 末影龙 ×1
3. 坚守者 ×1
4. 僵尸 ×10 + 骷髅 ×10
5. 苦力怕 ×10
6. 敌对铁傀儡 ×10
## 安装

1. 安装 Fabric API
2. 将模组 jar 文件放入 mods 文件夹
3. 启动游戏
